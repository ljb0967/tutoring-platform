# tutoring-platform
대학생 간 과외 플랫폼 아이디어를 구상하다가 실제 웹으로 옮겨보았습니다.
결론적으로 BM 구축하기가 쉽지 않고 충분한 수요가 확보되지 않을거 같아서 아이디어는 접었습니다.
- 서버단: node app.js (MongoDB 연결)
- 프론트단: npm start
